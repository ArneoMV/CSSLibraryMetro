Ovdije možemo vidjeti četiri primjera navigacije. Pritiskom na određenu hamburger ikonu otvaramo meni koji se ovisno o vrsti navigacije može prikazati sa lijeve, desne, vrha ili dna stranice.

Uputstva
  • sideNavigation klasa unutar koje se nalaze elementi navigacije. Napravljena je pomoću grida koji se sastoji od tri kolone. Prva kolona je za logo, druga za naslov, te treća za hamburger ikonu.
  • burger klasa služi kako bi animirala burger koji ima ima dva stanja. Napravljen je pomoću input type="checkbox" taga. Svaki input je povezan sa label tagom. Input sadrži id koji se povezuje sa for elementom. Input tag sadrži onclick=" " element koji služi kako bi povezali :checked stanje sa javascriptom koji izvršava funkciju prikazivanja sidebar-a
  • leftSidebar,rightSidebar,topSidebar i bottomSidebar klase služe kako bi prikazali sidebar. Napravljeni su pomoću @mixin sidebar koji im daje baznu strukturu. Kod ovih klasa zasebno se kodiraju height, width, top i bottom elementi kako bi se sidebar prikazao sa željene strane ekrana.
    Sidenote: sidebarovi su prilagođeni prikazu na ovoj stranici. Ukoliko želite korititi sidebar trebat ćete ih prilagoditi Vašoj main navigaciji.
  • leftSidebar,rightSidebar,topSidebar i bottomSidebar id-evi služe kako bi se sidebar povezao sa javascriptom koji računa koliko i gdje će se sidebar pomaknuti nakon pritiska na hamburher ikonu..
