Hamburger icon

The burger icon or burger menu is the name given to the menu icon found in newer programs and websites. It is one of the most popular navigation components. Serves as a visual representation of a button that displays and hides the menu. Using the hamburger menu makes it easier to view program options on mobile devices.

The burger consists of three horizontal lines. By clicking on the burger lines, the lines are animated in the X character. It contains two states: a hamburger icon when navigation is closed and an X icon when navigation is open.





class explanation
burger stylizes a space with three horizontal lines and X lines
bar stylizes horizontal lines
bar-1, bar-2, bar-3 separately position each horizontal line
  • The first icon was created using the input element. In order for the icon to be animated and transformed, it is necessary to use the type: 'checkbox' attribute to allow the change between two states within the icon.
  • The second icon was created using javascript to add or remove the "open" class for icon animation by clicking on the icon
