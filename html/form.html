Form

A form is a medium of a website that enables the input and processing of data by which we make contact with a user. With the help of backend programming languages ​​like NodeJS, PHP, Python and others, the form becomes functional and the collected data can be used for further processing.

The forms include elements such as: text box, radio button, checkbox button, checkbox, drop-down lists, data transfer and buttons.

The web form acts as a communication channel between the user and the website and vice versa. Illogical and inappropriate issues can very easily compromise the image of the company. For example, you want a user to open a profile on your website. Asking for a residence address would be completely irrelevant. The first thing a user would think would be: "What do you need my residence address for login only?". This creates confusion and distrust, and will increase the chance of a user not making a login. You should know your target audience to ask relevant questions.

To create a web form, you need to follow the guidelines to facilitate the user journey when interacting with the form.

  • Content is contained within one column
  • Align content vertically
  • Group the relevant information, and group the title with its input field
  • Avoid all capital letters
  • Show all selection options horizontally if below six options
  • Place checkboxes and radio buttons under each other for clarity
  • Make the description of the call to action button clearly described
  • Show the user where the error occurred and state the reason
  • Display help text
  • Show the difference between primary and secondary action
  • The size of the input field should be of relevant length with respect to the information being entered

class explanation
progress-bar class inside the form serves to see which multi-page form window we are on. It consists of progress-bar__dot, full, and progress-bar__connector classes that graphically regulate appearance. They are linked to javascript that specifies which form window we are in.
step class is linked by javascript. Serves to display each form window separately.
step1, step2, step3 within these classes, we insert the components that we want to display within each step of the form.
