Sidebar

We have created a navigation menu that, by clicking on the burger icon, is displayed on one of four sides.

class, id explanation
sideNavigation A class that contains navigation elements. It is made using a grid consisting of three columns. The first column is for the logo, the second for the title, and the third for the burger icon.
burger A class that serves to animate a burger that has two states. It was created using the input type ="checkbox" tag. Each input is associated with a label tag. The input contains an id that associates with the for element. The input tag contains an onclick = " element that is used to associate a checked state with Javascript that performs the sidebar rendering function.
leftSidebar,rightSidebar,topSidebar i bottomSidebar These classes serve to display the sidebar. They are made using the @mixin sidebar which gives them a base structure. These classes separately encode height, width, top and bottom elements to display the sidebar on the desired side of the screen. Sidenote: Sidebars are customized for the view on this page. If you want to use the sidebar you will need to adjust them to your main navigation.
leftSidebar, rightSidebar, topSidebar and bottomSidebar This id's serve to associate the sidebar with Javascript that counts how much and where the sidebar will move after clicking the burger icon.
