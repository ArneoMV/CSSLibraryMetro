Typography

Tipografija web stranice jedan je od najvažnijih dijelova web stranice. Kada korisnik otvori web stranicu prvo će primijetiti tipografiju. Ona treba biti čitka i razumna kako bi korisniku prenijela sve informacije koje korisnik treba u tom trenutku. Također treba paziti na kontrast između naslova i paragrafa kako u protivnom ne bi zbunili čitatelja.

Korisimo tri familije fontova.

  • Libre Baskerville je serifni font i koristimo ga za naslove.
  • Montserrat je sans-serifni i koristimo ga za paragrafe.
  • Ubuntu Mono je monospace font i koristimo ga za liste.

Fontovi zajedno daju vizualni sklad čitatelju jer imaju dobar kontrast . Čitatelj će moći razlikovati sekcije teksta dok će provoditi vrijeme na web stranici.

@font-face pravilo služi kako bi dohvatili fontove koji su dodani u naš root folder gdje smo spremili fontove. Jednom kada je opcija dodana u CSS, pravilo govori web pregledniku gdje treba dohvatiti font. Kako bi mrežni preglednik uspješno ispisao font, font treba biti WOFF2 (Web Open Font Format) ekstenziji. Ona služi kako bi spremili komprimirane podatke koji podržavaju TrueType (.ttf) i OpenType (.otf) ekstenzije.

@font-face { font-family: 'Montserrat'; src: url('../fonts/montserrat-webfont.woff2') format('woff2'), url('../fonts/montserrat-webfont.woff') format('woff'), url('../fonts/europaextended-webfont.ttf') format('truetype'); font-weight: normal; font-style: normal; } @font-face { font-family: 'UpheavalTT'; src: url('../fonts/upheavtt-webfont.woff2') format('woff2'), url('../fonts/upheavtt-webfont.woff') format('woff'), url('../fonts/europaextended-webfont.ttf') format('truetype'); font-weight: normal; font-style: normal; } @font-face { font-family: 'UbuntuMono'; src: url('../fonts/UbuntuMono/UbuntuMono-Regular.woff2') format('woff2'), url('../fonts/UbuntuMono/UbuntuMono-Regular.woff') format('woff'), url('../fonts/UbuntuMono/UbuntuMono-Regular.ttf') format('truetype'); font-weight: normal; font-style: normal; } @font-face { font-family: 'LibreBaskerville'; src: url('../fonts/LibreBaskerville/LibreBaskerville-Regular.woff2') format('woff2'), url('../fonts/LibreBaskerville/LibreBaskerville-Regular.woff') format('woff'), url('..fonts/LibreBaskerville/LibreBaskerville-Bold.ttf') format('truetype'); font-weight: normal; font-style: normal; } @font-face { font-family: 'LibreBaskerville-Italic'; src: url('../fonts/LibreBaskerville/LibreBaskerville-Italic.woff2') format('woff2'), url('../fonts/LibreBaskerville/LibreBaskerville-Italic.woff') format('woff'), url('..fonts/LibreBaskerville/LibreBaskerville-Italic.ttf') format('truetype'); font-weight: normal; font-style: italic; } @font-face { font-family: 'LibreBaskerville-Bold'; src: url('../fonts/LibreBaskerville/LibreBaskerville-Bold.woff2') format('woff2'), url('../fonts/LibreBaskerville/LibreBaskerville-Bold.woff') format('woff'), url('../fonts/LibreBaskerville/LibreBaskerville-Regular.ttf') format('truetype'); font-weight: bold; font-style: normal; } SCSS

  • <blockquote>tag blockquote koristi se za prikazivanje sadržaja iz drugog izvora
  • <q> tag služi za stavljanje navodnika.
  • <footer> tag prikazuje referencu teksta.
  • Selektor: nth-of-type (n) odgovara svakom elementu koji je n-to dijete određenog tipa, njegovog roditelja.
  • Selektor će svaki sljedeći (neparni) citat uvrstiti na desnu stranu stranice.
