Typography

We use three font families. The titles use the Libre Baskerville font, which is a serif font and looks antique, for paragraphs we use the Montserrat font, which is a sans-serif font and looks modern, and for lists use Ubuntu Mono, which is a monospaced font. Libre Baskerville, Montserrat and Ubuntu Mono fonts work visually in harmony with the contrast they create. The reader will be able to distinguish sections of the text while spending time on the website.

  • Libre Baskerville is a serif font and we use it for titles.
  • Montserrat is sans-serif and we use it for paragraphs.
  • Ubuntu Mono is a monospace font and we use it for lists.

The @font-face rule is used to retrieve the fonts that were added inside the root folder where we stored our fonts. When a font is added to CSS, the rule tells the web browser where to fetch the font. For a web browser to successfully print a font, the font must be stored with the Web Open Font Format (WOFF2) extension. WOFF2 serves to store compressed data that supports TrueType (.ttf) and OpenType (.otf) extensions.

@font-face pravilo služi kako bi dohvatili fontove koji su dodani u naš root folder gdje smo spremili fontove. Jednom kada je opcija dodana u CSS, pravilo govori web pregledniku gdje treba dohvatiti font. Kako bi mrežni preglednik uspješno ispisao font, font treba biti WOFF2 (Web Open Font Format) ekstenziji. Ona služi kako bi spremili komprimirane podatke koji podržavaju TrueType (.ttf) i OpenType (.otf) ekstenzije.
