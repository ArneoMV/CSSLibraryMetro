Colors

Metro allows the use of four colors to transmit information: blue (primary), orange (warning), red (danger), and green (success). Also, we use white and black that does not fit into the spectrum of the other four colors.

  • Blue - openness, intelligence, faith, stability, clarity
  • Orange - active, happiness, enthusiasm, creativity, appetite
  • Red - aggression, passion, exciting, dramatic, love
  • Green - nature, calming, growth, freshness, hope, wealth, education
  • White - purity, perfection, light
  • Balck - death, evil, power, elegance

Colors are defined using variables that can be used within SCSS. Within each color group, a palette of five different color tones was made. To use color, it is necessary to write its variable to the location of the value that is intended to display the color. If there is a need to change color within the Metro work environment, this can be easily accomplished within the "_colors.scss" folder.
