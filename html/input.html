Input

The HTML <input> selector is used to create interactive web forms. They are used to accept information from users. The input accepts a variety of attribute values. Depending on the value of the attribute, the input element will perform the associated actions. If the attribute is not specified, the default attribute value is text.


Input types

How an <input> selector works varies depending on the value of its type attribute. Here are some of the basic attribute types.


Type Description
text The default value. A single-line text field. Line-breaks are automatically removed from the input value.
date A control for entering a date (year, month, and day, with no time) opening a date picker or numeric wheels for a month, hour, etc., when active in supporting browsers.
number A control for entering a number; displays a spinner and adds default validation when supported. Displays a numeric keypad in some devices with dynamic keypads.
checkbox A checkbox allowing single values to be selected/deselected.
submit A button that submits the form.


Attributes

Attributes are a powerful part of the <input> element. Each <input> element is based on an HTMLInputeElement interface, so technically all <input> elements share exactly the same set of attributes. However, in reality, most attributes only affect a certain subset of input types.

Attribute Type Description
required almost all The boolean value is required or must be check for the form to be submitted.
pattern password, text, tel Pattern the value must match to be valid.
checked radio, checkbox Whether the command or control is checked.
form all Associates the control with a form element.
placeholder password, search, tel, text, URL Content that will be displayed inside the input element to indicate what follows next.
