Shadow

The shadow effect places the shadow on the frame of the element.

For the CSS framework, we have created four classes for using shadows: shadow-none, shadow, shadow-sm, and shadow-lg. The shadow-none class removes shadows from an element, while the other classes differ in the amount of shadow placed on an element.


An explanation of the attributes that affect the shadow

attributes explanation
Offset-x Sets the horizontal offset of the shadow
Offset-y Sets the vertical offset of the shadow
Blur radius Sets the blur radius
Color Sets the color of the shadow

shadow-sm The class that adds a small size to the shadow
noshadow The class that excludes element shadow
