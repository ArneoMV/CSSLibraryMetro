Images

Uputstva
  • <img> oznaka koristi četiri atributa: srcset, sizes, src, te alt.
  • Unutar HTML koda koristili smo <figure> oznaku koja služi kako bi dodali element <figcaption> koji nam omogućuje ispis naslova slike ili medija
  • <src> služi kako bi našli root do slike koju prikazujemo
  • <alt> služi kako bi ispisao naziv slike ukoliko sliku ne možemo prikazati na ekranu
  • <srcset> atribut služi kako bi dohvatili set slika za različite slike ekrana unutar
<figure> <img srcset="../img/slika320.png 320w, ../img/slika480.png 480w, ../img/slika720.png 800w" sizes="(max-width: 320px) 280px, (max-width: 480px) 440px, 800px" src="../img/slika320.png" alt="slika"> <figcaption>slika 320wpx</figcaption> </figure> HTML

Proširivanjem CSS koda sa iframe, object, embed, te video omogučujemo responzivnu razlučivot ostalih media elemenata.

img, iframe, object, embed, video { width: 100%; max-width: 100%; } SCSS
