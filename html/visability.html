Visability

Korištenje klasa

Unesite sljedeće klase kako bi prikazali i/ili sakrili element. Ove klase ne mijenjaju prikaznu vrijednost i ne utječu na izgled - elementi koji nisu vidljivi još uvijek zauzimaju prostor na stranici.

Uputstva
  • visible klasa koja prikazuje element.
  • invisible klasa koja skriva element.

Bok, ja sam skriven. Primijetite da je i sav moj stil skriven i da još uvijek zauzimam prostor iako me ne možete vidjeti.
Bok, moj roditeljski element je skriven, ali ja sam još uvijek vidljiv. Prođite mišem iznad mene da bi roditelj bio vidljiv.
<div class="visible">...</div> <div class="invisible">...</div> HTML
/* VISABILITY */ // Class .visible { visibility: visible; } .invisible { visibility: hidden; } // Usage as a mixin @mixin invisible($visibility) { visibility: $visibility !important; } .element { @include invisible(visible); } .element { @include invisible(hidden); } SCSS
