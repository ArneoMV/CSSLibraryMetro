Border's

The borders serve to add an edge to the element in any of the colors Metro offers.

The thickness of the borders is 1px. The border color can be one of the following: blue, orange red, green, white or black.



class explanation
border class gives the thickness of the boundary line. Line thickness is 1px.
border-primary class specifies the color of the border that is the primary color
border-warrning class specifies the color of the border that is the warrning color/td>
border-danger class specifies the color of the border that is the primary color
border-success class specifies the color of the border that is the danger color
border-white class specifies the color of the border that is the white color
border-black class specifies the color of the border that is the black color
