Tooltip

A tooltip is a graphical user interface element used in conjunction with the cursor or mouse pointer to display information about an item without needing to click on it

A tooltip is also known as a hint, infotip or screentip.

We add the Tooltip to an element using the associated class. The text we want to present findings The text that we want to display is called by a data attribute named "tooltip", the data attribute "tooltip" is linked to the data attribute "flow" by which we determine where the text will be displayed.

class explanation
left class shows the text on the left of the element
right class displays the text to the right of the element
up class displays the text above the elements
down class shows the text below the element
left-down class displays the text in the lower-left corner of the element
right-down class displays the text in the right-left corner of the element
left-up class displays the text in the upper-left corner of the element
right-up class displays the text in the upper-right corner of the element
