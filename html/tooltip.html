Tooltip

Što je opis u HTML kodu?
Atribut "title" može odrediti tekst koji će se pojaviti kao opis pri prelasku miša preko slike. Upotrijebite ovo za usmjerivanje alatnih znakova na ikoni ili bilo kojoj drugoj mrežnoj slici. Alt tekst se prikazuje ako se slika iz nekog razloga ne može prikazati, primjerice ako je preglednik ne može učitati od mjesta na kojem je pohranjena.

Osnovni opis
HTML: Upotrijebite element spremnika (poput <div>) i dodajte mu klasu "tooltip". Kad korisnik pređe mišem preko tog <divgt;, prikazat će se opis teksta. Opisni tekst nalazi se unutar umetnutog elementa (poput <span>) s class = "tooltiptext"

Što je tooltip u JavaScript-u?
Pregled. HTML5 / JavaScript Tooltip komponenta je skočni prozor koji prikazuje informacije ili poruku kada korisnici lebde, kliknu, fokusiraju ili dodirnu sliku, gumb, oznaku za sidro itd. Podaci prikazani u Tooltipu mogu uključivati jednostavan tekst, slike, hiperveze ili prilagođene predloške.
