Top menu

Top menu navigation serves to have a navigation menu at the top of the page. It is a responsive design and features two levels of drop-down menus.


class explanation
navBar class within which I style all the components of a <nav> tag
toggle with this class we open dropdown on the elements
drop, drop-1, drop-2, drop-3 each dropdown has its own serial number. The for and id element are linked using a name. It is used to know which dropdown should be opened.
