Uputstva

Progres bar napravljen od <progress> taga koji je uključen u HTML5

  • <progress> element može se koristiti zajedno s JavaScriptom za prikaz napretka zadatka ili procesa onako kako je u tijeku.
  • <progress> oznaka obično se piše kao <progress value= "0" max= "100"> </progress> s bilo kojim sadržajem unutar početne i završne oznake. Taj sadržaj može biti trenutna vrijednost i / ili maksimalna vrijednost koja se može prikazati naslijeđenim preglednicima (tj. preglednicima koji ne podržavaju element <progress>).
  • Atribut vrijednosti predstavlja trenutnu vrijednost (tj. gdje je napredak trenutno na). Ako izostavite taj atribut, traka napretka postaje "neodređena" i neće prikazati nikakav napredak (ali većina preglednika prikazivat će animiranu traku koja će pokazati korisniku da pokušava ...).
  • Atribut max predstavlja ukupni iznos (tj. gdje će vrijednost trake napretka biti kad se završi).

<progress> tag obično je vezan za Ajaxov prijenos događaja.

// Progress bar -> progress var milisec = 0; var max = 100; function displayProgress(){ if(milisec >= max){ document.getElementById('message').innerHTML= "Done!"; } milisec += 10; document.getElementById('downloadProgress').value = milisec; setTimeout("displayProgress()",max); } JS
